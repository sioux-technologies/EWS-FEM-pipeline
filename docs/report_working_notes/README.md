# Report Working Notes

This folder contains compact handover notes and supporting report-development material for the COMSOL EWS breast FEM project.

Start here:

1. `parameter_overview.md`  
   Compact overview of the main TOML-style model parameters and the reference values used in the project.

2. `current_limitations.md`  
   Main modelling limitations, interpretation caveats, and recommended next steps.

3. `pipeline_notes/comsol_pipeline_guide.md`  
   Practical guide for running and post-processing COMSOL cases.

## Folder Overview

- `literature_overviews/`  
  Literature-based context for material parameters, breast volume, glandular fraction, and stress/displacement interpretation.

- `model_justification/`  
  Notes explaining why specific modelling choices were made.

- `pipeline_notes/`  
  Technical notes on COMSOL pipeline usage, exports, and manual CSV workflows.

The final report and presentation are stored in `docs/internship_deliverables/`.
