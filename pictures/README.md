# Pictures

This folder contains project figures, model screenshots, schematics, and short animation exports used for documentation, reporting, and presentation preparation.

The main COMSOL pipeline does not depend on these files. They are included as visual reference material for the model-development workflow.

## Folder Overview

- `general/`  
  General explanatory figures, including dynamic loading, material stiffness, Mooney-Rivlin, and displacement examples.

- `previous_work/`  
  Figures related to the previous FEBio-based EWS breast-model work.

- `mesh/`  
  Mesh screenshots and report-ready mesh views.

- `model_overviews/`  
  Overview figures for the COMSOL model-development route and pipeline.

- `stage1_motion_baseline/`  
  Motion-baseline and early dynamic loading visuals.

- `stage2_chestwall/`  
  Chestwall curvature schematics and model screenshots.

- `stage3_glandular/`  
  Fibroglandular tissue distribution variants and reference views.

- `stage4_asymmetry_nipple/`  
  Outer-shape and nipple-position variation examples.

- `stage5_cooper/`  
  Cooper-like support schematics, stress screenshots, and selected animation exports.

- `stage6_tumor/`  
  Tumor-overlay placement examples for different sizes and locations.
